# RCS1036: Remove redundant empty line

Property | Value
--- | ---
Id|RCS1036
Category|Redundancy
Default Severity|Info
Enabled by Default|yes
Supports Fade\-Out|no
Supports Fade\-Out Analyzer|no

## Examples

### Code with Diagnostic

```csharp
public class Foo
{

    private string _f;


    public bool Bar()
    {

        return false;

    }

}
```

### Code with Fix

```csharp
public class Foo
{
    private string _f;

    public bool Bar()
    {
        return false;
    }
}
```

## How to Suppress

### SuppressMessageAttribute

```csharp
[assembly: SuppressMessage("Redundancy", "RCS1036:Remove redundant empty line.", Justification = "<Pending>")]
```

### \#pragma

```csharp
#pragma warning disable RCS1036 // Remove redundant empty line.
#pragma warning restore RCS1036 // Remove redundant empty line.
```
