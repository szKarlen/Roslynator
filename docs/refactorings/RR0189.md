## Reduce if nesting

Property | Value
--- | ---
Id|RR0189
Title|Reduce if nesting
Syntax|if statement
Span|if keyword
Enabled by Default|yes

### Usage

#### Before

```csharp
if (condition1)
{
    Foo1();

    if (condition2)
    {
        Foo2();

        if (condition3)
        {
            Foo3();
        }
    }
}
```

#### After

```csharp
if (!condition1)
{
    return;
}

Foo1();

if (!condition2)
{
    return;
}

Foo2();

if (!condition3)
{
    return;
}

Foo3();
```
