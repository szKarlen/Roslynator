## Replace object creation with default value

Property | Value
--- | ---
Id|RR0185
Title|Replace object creation with default value
Syntax|object creation expression
Enabled by Default|yes

### Usage

#### Before

```csharp
var x = new object();
```

#### After

```csharp
object x = null;
```
_____
#### Before

```csharp
var arr = new object[0];
```

#### After

```csharp
object[] arr = null;
```
